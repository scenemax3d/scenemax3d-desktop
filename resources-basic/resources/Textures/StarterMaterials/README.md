Starter material texture library for the SceneMax material builder.

Each starter pack lives in its own folder and follows the same layout:

- `diffuse.png`
- `normal.png`
- `glow.png`

This keeps the material-authoring workflow predictable when you browse for texture slots.

Included packs:

- `BrickWall`, `Pond`, `Rock`, `RockAlt`, `Dirt`, `Grass`, `Road`
- `BackroomsCarpet`, `BackroomsWall`, `BackroomsCeilingTile`, `BackroomsCeilingLight`
- `MarbleWhite`, `MarbleCream`, `MarbleVein`
- `SquareTileGreen`, `BrownTiles`
- `FabricWeaveWhite`, `FabricCarpetRed`, `FabricStripeMono`
- `LeatherBlack`, `LeatherBrown`, `LeatherStitched`
- `MetalPaintedBlack`, `GravelPebbles`
- `PlanksLight`, `PlanksWeathered`, `WoodFloorParquet`

Notes:

- Several packs are curated from textures already bundled with this repository.
- The `Backrooms*` packs come from the free GitHub repository `amini-allight/backrooms-textures`.
- That repository states the textures are licensed under `CC0 1.0`.
- The newer `Marble*`, `Tile*`, `Fabric*`, `Leather*`, `Metal*`, `Gravel*`, `Planks*`, and `WoodFloor*` packs are curated from `ambientCG`.
- `ambientCG` publishes these assets under `CC0 1.0`.
- Each curated ambientCG folder includes a local `SOURCE.txt` file with the original asset page.
- For non-emissive sets, `glow.png` was derived locally to give material artists a quick starting point for stylized emissive looks.
