# Minimap

## Show Minimap

Display a small map of the scene:

```scenemax
minimap.show
```

Set the minimap camera height (in meters):

```scenemax
minimap.show : height 100
```

Set the minimap display size (in pixels):

```scenemax
minimap.show : size(300,300)
```

Make the minimap follow a specific object:

```scenemax
Minimap.show : follow player1
```
