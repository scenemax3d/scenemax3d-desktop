# Effects

## Explosion

Show an explosion effect at an object's position:

```scenemax
effects.explosion.show : pos (b)
```

## Flame

Show a flame effect with a radius and duration:

```scenemax
effects.flame.show : pos (13,-15,43) and radius = 5 and duration 10
```

## Time Orbit

Show an orbit effect:

```scenemax
effects.TimeOrbit.show : pos (0,0,0) and end size = 7 and duration = 10
```
