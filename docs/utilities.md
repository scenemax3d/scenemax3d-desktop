# Utilities

## Wait

Pause execution for a specified duration:

```scenemax
wait 10 seconds
```

## Distance

Measure the distance between two objects:

```scenemax
var dist = distance(n,n2)
```

## Angle

Calculate the angle between two objects:

```scenemax
var ang = angle(n1,n2)
```

## Random Numbers

Generate a random number (0 to N-1):

```scenemax
var random_obj = my_objects[rnd(4)]
```

## Debug

Turn debug display on/off:

```scenemax
debug.on
debug.off
```
