# Screen & Canvas

## Full Screen Mode

Run the game in full screen mode:

```scenemax
Screen.mode full
```

## Canvas Size

Set the game canvas dimensions (width, height in pixels):

```scenemax
Canvas.size 1600,900
```

## Skybox

Load a background skybox for the game:

```scenemax
skybox.show "lagoon"
```

Load a solar system skybox:

```scenemax
skybox.show solar system
```

Set the time of day in the solar system skybox (24-hour format):

```scenemax
skybox.hour=12
```
